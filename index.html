Dynamic UI Design

Javascript is an integral part of modern web design and is used to create complex web applications that work across all devices and browsers. Many of the most basic website components we interact with on a daily basis are brought to life with Javascript.

This site serves as a demo to three of the most common interactive Javascript elemnts: the drop-down menu, mobile menu, and image slider.

For more information and links to the npm modules, please checkout the Github repo.
